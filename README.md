# О проекте
 Дипломный проект, в котором реализован блоговый движок в соответствии с документацией API.
 
# Используемые технологии
1. Java 8/11
2. Git
3. Spring Boot
4. Spring Security
5. MySQL 
6. Hibernate ORM
7. Cервис Cloudinary
